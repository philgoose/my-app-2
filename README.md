# my-app-2

메모장 웹 애플리케이션

## 기능
- 📝 메모 작성, 수정, 삭제
- 🔍 메모 검색
- 💾 자동 저장
- 📱 모바일 반응형 디자인
- 🌙 다크 모드 지원

## 설치 방법

```bash
# 저장소 클론
git clone https://github.com/philgoose/my-app-2.git
cd my-app-2

# 필요한 패키지 설치
pip install -r requirements.txt
```

## 실행 방법

```bash
python app.py
```

브라우저에서 `http://localhost:5000` 으로 접속하세요.

## 모바일 접근
같은 네트워크에서 PC의 IP 주소로 접속 가능합니다:
- `http://<your-pc-ip>:5000`
