# Memo App

## Firebase Database Setup

1. Go to [Firebase Console](https://console.firebase.google.com/).
2. Create a new project or select an existing one.
3. Navigate to the Database section.
4. Click on 'Create Database'.
5. Choose 'Start in Test Mode' for ease of development.
6. Add the necessary collections and documents as per your app's requirements.

## GitHub Pages Deployment

1. Push your code to the `main` branch of your GitHub repository.
2. Go to the repository settings.
3. Scroll down to the 'GitHub Pages' section.
4. Select `main` branch as the source and click 'Save'.
5. Your site will be published at `https://<username>.github.io/my-app-2/`.

## Installation Instructions

1. Clone the repository using `git clone https://github.com/philgoose/my-app-2.git`.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Start the application using `npm start`.
5. Open your browser and go to `http://localhost:3000` to access the app.
